# Kotlin Console Calculator

 Overview
A simple console-based calculator written in Kotlin that performs basic arithmetic operations.  
This project demonstrates Kotlin fundamentals such as functions, conditionals, loops, and error handling.

 Features
- Addition, Subtraction, Multiplication, and Division
- Input validation and error handling
- Modular code with reusable functions
- Graceful exit and clear UI messages

 Requirements
- Kotlin 1.9+ or IntelliJ IDEA / Android Studio
- JDK 17+
